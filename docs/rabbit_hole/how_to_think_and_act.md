# 关于微博
今天我花了三个小时试图搞懂如何订阅我关注的一个微博作者，但是失败了。

主要是因为我对底层网络运行、浏览器之间的攻防了解不足，但我意识到另一个很严重的问题，我完全没有必要再在微博这种平台上浪费时间，这并不是讽刺贬低微博，而是指，这个作者在微博提供的信息不足以让我浪费时间在探究微博攻城战。

移动到自由地，在封闭系统下所拥有的信息，在自由地必然存在，反之必然不成立，且移动到自由地是成本最低的操作。

同样，我重拾RSS这种原始的工具，在一个高效的环境下，效率的价值才真的体现出来。这是一个很诡异的选择，就是那种几十年的所谓抗拒努力都是白费，能做的只是更珍惜自己所拥有的时间。
