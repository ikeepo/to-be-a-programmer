# Intro 
[ddd-crew/welcome-to-ddd](https://github.com/ddd-crew/welcome-to-ddd)
每一个Domain Class has corresponding Config Class, 好处是：

1. 分离
   Domain类关注业务，Config关注配置；
2. 可扩展
   分离这个概念，自带可扩展。
   分离后各自都便于扩展。
