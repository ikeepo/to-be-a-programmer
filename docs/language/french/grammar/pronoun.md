# 代词-Pronoun
## Abstract

## 代词范式（paradigme pronominal）

```shell
第一人称单数
├── 主语位置 → je
├── 动词前宾语 → me
└── 独立/强调/命令式后 → moi

```
## 主语代词
主语代词不能单独站立。


## 反身代词 (Reflexive Pronouns)
表示动作的发出者和接受者是同一个人。
### Rules
#### 反身代词放在动词的前面
大部分情况。
功能层问题。
#### 反身代词很多时候'省音
`'`[ 是省音符号 ](./voice/elision.md#defination)，跟反身代词概念没有直接关系，只要是 “单音节词 + 以 -e 结尾”，在遇到元音或哑音 H 时，几乎都要服从这个协议.

只是反身代词大多都是e结尾，所以比较明显.

## 重读代词 (Stressed Pronouns)
主语代词（如 Je, Tu）是很弱的，不能单独站立；而重读代词则是强力插件，专门用于需要“**加重语气**”或“**独立支撑**”的结构中。

### List

| 人称 | 主语代词 (Subject) | 重读代词 (Stressed) | 常用翻译 (Meaning) |
| :--- | :--- | :--- | :--- |
| **第一人称单数** | *Je* | **Moi** | Me |
| **第二人称单数** | *Tu* | **Toi** | You |
| **第三人称单数 (阳)** | *Il* | **Lui** | Him |
| **第三人称单数 (阴)** | *Elle* | **Elle** | Her |
| **第一人称复数** | *Nous* | **Nous** | Us |
| **第二人称复数** | *Vous* | **Vous** | You |
| **第三人称复数 (阳)** | *Ils* | **Eux** | Them |
| **第三人称复数 (阴)** | *Elles* | **Elles** | Them |

### 应用场景

| 应用场景 (Scenario) | 逻辑说明 (Logic) | 错误用法 (Wrong) | 正确用法 (Right) |
| :--- | :--- | :--- | :--- |
| **1. 肯定命令句后** | 宾语代词后置时必须强化。 | *Regarde-me (X)* | **Regarde-moi !** (看着我) |
| **2. 介词之后** | 介词（如 avec, pour）后须用重读。 | *Avec tu (X)* | **Avec toi** (和你一起) |
| **3. 强调或对比** | 用于突出主语或进行人称对比。 | *Je, j'aime... (X)* | **Moi, j'aime...** (我嘛，我喜欢...) |
| **4. 独立/简短回答** | 无动词支撑时，代词须能独立生存。 | *Qui ? Je. (X)* | **Qui ? —— Moi.** (谁？—— 我。) |
