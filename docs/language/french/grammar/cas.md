# 格Cas
## Abstract
格是语法上的一种功能标记，用来显示名词、代词或形容词在句子中的句法功能。

格告诉我们一个词在句中是作 主语、宾语、间接宾语，还是用于 介词/强调。

现代法语主要关注 主格 vs 宾格/重读代词。

格明确了句子关系（谁做动作，动作影响谁/什么），保证语序、语态和主客关系合理。

## pronom tonique（重读代词）
在现代法语里并不叫“宾格代词”。它们的正式名称是：pronom tonique（重读代词）

特征是可以被重读、可以单独存在（不用必须在动词前）；

## 代词范式（paradigme pronominal）
```shell
第一人称单数
├── 主语位置 → je
├── 动词前宾语 → me
└── 独立/强调/命令式后 → moi
```
