# 句子复杂度 Sentence Complexity
四大句型type每种都可以包含从句。

复杂度是句子的结构层。


### 法语从句全架构整合表
从句的类型由 从句词 决定，可以与各种句型搭配。

| 语法属性 | 从句名称 | 引导词 (Marker) | 角色定位 (Role) | 经典实例 |
| :--- | :--- | :--- | :--- | :--- |
| **名词性** | **宾语从句** | **que** | 充当动词的宾语 | *Je sais **qu'il vient**.* |
| (Noun-like) | **主语从句** | **que** | 充当句子主语 | * **Qu'il vienne** est vrai.* |
| **形容词性** | **定语从句** | **qui / que** | 修饰/限定名词 | *Le livre **que je lis**.* |
| (Adj-like) | | **dont / où** | 修饰所属或地点 | *Le pays **où je suis**.* |
| **副词性** | **条件从句** | **si** | 设定前提条件 | ***S'il te plaît**...* |
| (Adv-like) | **时间从句** | **quand** | 交代时间背景 | *Je sors **quand il pleut**.* |
| | **原因从句** | **parce que** | 交代因果逻辑 | *Il rit **parce que c'est drôle**.* |
| | **目的从句** | **pour que** | 交代行动目标 | *Lisez **pour que vous appreniez**.* |
