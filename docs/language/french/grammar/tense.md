# 时态 Tense

## 时态类型

| 优先级 | 时态名称 | 身份标识 (Identifier) | 核心逻辑 | 经典实例 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **现在时** | 词尾变位 (-e, -s, -t) | 现状、真理、习惯 | *Je mange.* |
| **2** | **复合过去时** | **avoir/être + 过去分词** | 已完成的点状动作 | *J'ai mangé.* |
| **3** | **最近将来时** | **aller + 动词原形** | 即将发生的计划 | *Je vais manger.* |
| **4** | **未完成过去时** | 词缀 **-ais, -ais, -ait** | 过去的背景/状态 | *Je mangeais.* |
| **5** | **简单将来时** | **r + -ai, -as, -a** | 预测或遥远的未来 | *Je mangerai.* |
| **6** | **最近过去时** | **venir de + 动词原形** | 刚刚发生的动作 | *Je viens de manger.* |
| **7** | **愈过去时** | **avais/étais + 过去分词** | 过去的过去 | *J'avais mangé.* |

### 三：最近将来时
表示即将发生或意图明确的动作。

最近将来时跟`aller`深度绑定，`aller(变位) + 动词原形`
