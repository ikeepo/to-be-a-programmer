# 形容词 Adjective

### 法语物主形容词表 (Possessive Adjectives)
对于“物主形容词”本身，复数确实不分阴阳；但对于“普通形容词”，复数依然要看阴阳。

| 人称 | 阳性单数名词 | 阴性单数名词 | 复数名词 (不论阴阳) |
| :--- | :--- | :--- | :--- |
| **Je** (我的) | *mon* | *ma* | *mes* |
| **Tu** (你的) | *ton* | *ta* | *tes* |
| **Il/Elle** (他/她的) | *son* | *sa* | *ses* |
| **Nous** (我们的) | *notre* | *notre* | *nos* |
| **Vous** (你们/您的) | *votre* | *votre* | *vos* |
| **Ils/Elles** (他们的) | *leur* | *leur* | *leurs* |

**⚠️ 特殊规则 (Special Rule)：**
如果阴性单数名词以**元音**或**哑音 h** 开头，为了语音顺滑，必须使用阳性形式。
* *Ma amie* (❌) -> ***Mon** amie* (✅)
