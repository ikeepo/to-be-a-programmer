# Intro 

| 层级 (Level) | 名称 | 逻辑角色 | 类比 (Analogy) |
| :--- | :--- | :--- | :--- |
| 第四层 (应用/表现) | 语流 (Speech Stream) | 最终 the 物理产出。 | 渲染出的视频画面 |
| 第三层 (规则/逻辑) | 语法 (Grammar) | 决定结构与形态的法律。 | 视频的剪辑逻辑与脚本 |
| 第二层 (声音系统) | 音位 (Phonology) | 决定声音如何组合的协议。 | 编码器/解码协议 |
| 第一层 (底层素材) | 词汇 (Lexicon) | 原始的单词素材仓库。 | 原始素材素材包 |
