# 连诵 Liaison
`词 A (结尾辅音不发音) + 词 B (开头是元音/哑音 h) = 连起来读`

## 为什么连诵
避免间断Hiatus：两个元音连在一起
## 连诵时候，前面单词的音要改变么？

### nazalized vowels change
去鼻化或者弱化。
`un ami [œ̃.na.mi] 或 [ɛ.na.mi]` 原本鼻化的元音 [œ̃] 会变得比较“干巴巴”，更接近普通的元音 [ɛ] 或 [œ]，然后接上一个清晰的 [n]。
### 普通单词不变
只是末尾的辅音复活了
### 有些音变

### 🇫🇷 法语联诵音变对照表 (Liaison)

| 单词结尾字母 | 单独读（末尾不发音） | 联诵时的声音 (IPA) | 典型例子 | 实际联诵读音 |
| :--- | :--- | :--- | :--- | :--- |
| **-s** | (无声) | **[z]** | *le**s** amis* (朋友们) | [le.**z**a.mi] |
| **-x** | (无声) | **[z]** | *deu**x** ans* (两年) | [dø.**z**ɑ̃] |
| **-d** | (无声) | **[t]** | *quan**d** il...* (当他...) | [kɑ̃.**t**il] |
| **-g** | (无声) | **[k]** | *san**g** impur* (不纯的血) | [sɑ̃.**k**ɛ̃.pyʁ] |
| **-f** | (无声/或发[f]) | **[v]** | *neu**f** ans* (九岁) | [nœ.**v**ɑ̃] |


* **S/X 浊化为 [z]：** 这种变化最常见，发音时带声带振动。
* **D/G 清化为 [t]/[k]：** 这里的逻辑是为了让气流更顺畅地滑向下一个元音。
* **F 变 [v]：** 这种情况比较少见，主要出现在数字 `neuf` (9) 后面接 `ans` (岁) 或 `heures` (小时) 时。

### 💡 核心避坑指南 (Markdown Check-list)

* [ ] **[s] / [x] 变 [z]**：*Les amis*, *Deux ans*.
* [ ] **[d] 变 [t]**：*Quand il pleut*, *Grand enfant*.
* [ ] **[f] 变 [v]**：*Neuf ans*, *Neuf heures*.
* [ ] **鼻化元音弱化**：*Un enfant* [œ̃.nɑ̃.fɑ̃]，元音会稍微变“扁”。
* [ ] **绝对禁区**：看到 **"et"** (和) 永远不要连读！
