## les repas sont préparés et consommés 为什么这里需要sont，直接名词+过去分词作形容词修饰不可以么
是可以的。

| 结构                      | 类型    | 重点           |
| ----------------------- | ----- | ------------ |
| repas préparés          | 形容词用法 | 状态（已经做好的饭）   |
| les repas sont préparés | 被动语态  | 动作（饭被制作这个过程） |
