# voir 动词

| 动词 (Verb) | 含义 (Meaning) | 类别 (Category) | 备注 (Notes) |
| :--- | :--- | :--- | :--- |
| {{fr('voir')}} | 看见 (To see) | 视觉类 (基础) | 核心动词，变位极常用。 |
| {{fr('revoir')}} | 再次见到 (To see again) | 视觉类 (派生) | 常见于 Au revoir。 |
| {{fr('entrevoir')}} | 瞥见 / 隐约看见 (To glimpse) | 视觉类 (派生) | 指不完全或短促的看见。 |
| {{fr('prévoir')}} | 预见 / 计划 (To foresee) | 视觉类 (派生) | 注意：简单将来时为 prévoirai (词干不缩写)。 |
| {{fr('pourvoir')}} | 提供 / 配备 (To provide) | 视觉类 (派生) | 常用搭配：pourvoir à (满足/负责)。 |
| {{fr('dépourvoir')}} | 剥夺 / 使缺乏 (To deprive) | 视觉类 (派生) | 常用于被动语态 être dépourvu de。 |
| {{fr('avoir')}} | 有 (To have) | 助动词 | 法语最核心动词之一，地位特殊。 |
| {{fr('devoir')}} | 必须 / 欠债 (To must / owe) | 情态动词 | 过去分词为 dû (带长音符以区分 du)。 |
| {{fr('pouvoir')}} | 能够 (To be able to) | 情态动词 | 表示可能性、许可或能力。 |
| {{fr('savoir')}} | 知道 / 会 (To know) | 情态动词 | 表示掌握知识、事实或技能。 |
| {{fr('vouloir')}} | 想要 (To want) | 情态动词 | 虽结尾稍异，但变位逻辑高度相关。 |
| {{fr('recevoir')}} | 接收 (To receive) | 感知类 (基础) | 变位遇到 o/u 时 c 变 ç (如 je reçois)。 |
| {{fr('apercevoir')}} | 察觉 / 瞥见 (To perceive) | 感知类 (派生) | s'apercevoir de 表示“意识到”。 |
| {{fr('concevoir')}} | 构思 / 怀孕 (To conceive) | 感知类 (派生) | 指精神上的设计或生理怀孕。 |
| {{fr('décevoir')}} | 使失望 (To disappoint) | 感知类 (派生) | 过去分词为 déçu。 |
| {{fr('percevoir')}} | 感知 / 收取 (To perceive / collect) | 感知类 (派生) | 用于感官感知或财务收税。 |
| {{fr('valoir')}} | 值得 (To be worth) | 功能类 | 常用句型：Il vaut mieux... (最好...)。 |
| {{fr('mouvoir')}} | 移动 (To move) | 功能类 | 较为正式，现代法语多用 bouger。 |
| {{fr('émouvoir')}} | 使感动 (To move emotionally) | 功能类 (派生) | 变位与 mouvoir 类似。 |
| {{fr('promouvoir')}} | 晋升 / 促销 (To promote) | 功能类 (派生) | 过去分词为 promu。 |
| {{fr('pleuvoir')}} | 下雨 (To rain) | 无人称动词 | 仅使用第三人称单数 (Il pleut)。 |
| {{fr('falloir')}} | 必须 / 需要 (To be necessary) | 无人称动词 | 仅使用第三人称单数 (Il faut)。 |
| {{fr('asseoir')}} | 使坐下 (To seat) | 特殊类 | 有两套变位系统 (j'assieds / j'assois)。 |
| {{fr('seoir')}} | 适合 (To suit) | 特殊类 | 现多用于分词形式 séant 或第三人称。 |
