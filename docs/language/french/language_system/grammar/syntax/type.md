# 句类 Type

## 四大句型
### 陈述句
#### 元素结构顺序

| 顺序 | 成分名称 (Component) | 实例 (Example) | 逻辑功能 (Function) |
| :--- | :--- | :--- | :--- |
| **1** | **主语 (Subject)** | **Je** | 动作的发起者，系统的核心。 |
| **2** | **助动词/半助动词** | **vais** | 承载变位，标记**时间层**。 |
| **3** | **核心动词原形** | **prendre** | 承载动作的具体语义。 |
| **4** | **直接宾语 (COD)** | **un kilo de...** | 动作的直接承受对象。 |
| **5** | **状语 (Adjunct)** | **(如有) au marché** | 补充地点、时间、方式。 |

#### 名词句/无动词句
以名词为核心，省略了动词；

信息高度压缩	常见于：标题、广告、医嘱、电报、标签；

### 疑问句

#### 元素结构顺序

| 阶级 | 架构类型 | 结构顺序 | 实例 (Example) | 逻辑本质 (Logic) |
| :--- | :--- | :--- | :--- | :--- |
| **低 (口语)** | **语调式** | **陈述句原序 + ?** | *Tu vas prendre un kilo ?* | 最简单，仅靠语调提升。 |
| **中 (通用)** | **Est-ce que 式** | **Est-ce que + 陈述句原序** | *Est-ce que tu vas prendre un kilo ?* | 插入一个疑问标识符，语序不变。 |
| **高 (正式)** | **倒装式** | **变位动词 - 主语 + 核心动作** | *Vas-tu prendre un kilo ?* | 语序反转，最具书面感。 |



### 命令句/祈使句

#### 元素结构顺序

| 顺序 | 成分名称 (Component) | 实例 (Example) | 逻辑功能 (Function) |
| :--- | :--- | :--- | :--- |
| **1 (消失)** | **主语 (Subject)** | **(Tu / Nous / Vous)** | **强制省略**：主语不出现在句子中。 |
| **2** | **变位动词** | **Prends / Prenez** | **核心指令**：使用命令式变位。 |
| **3** | **直接宾语 (COD)** | **un kilo de...** | 指令的对象。 |
| **4** | **状语/修饰 (Adjunct)** | **bien mûres** | 补充细节。 |

#### 动词变位规则例外

`Ferme la porte.` 为什么 Ferme 没有 s？在法语词法形变 (Morphology) 规则中，所有以 -er 结尾的动词，[在命令式单数形式下都要去掉 `s`](./conjugation.md#变位规则例外-命令式).

- 为什么要有去掉`s`这个规则?
```shell
1. Etymology： 拉丁语如此

2. Visual Distinction: 动词变位发音基本一致，去掉s可以从视觉上体现orthography

3. Liguistic Economy: 命令式要求极速, 前省主语，后省s
```

#### 主语显化原则 (Principle of Explicit Subject) vs. 命令式豁免 (Imperative Exception)

在法语的底层架构中，主语显化是为了区分变位。因为法语动词在很多时候发音是一样的（比如 ferme, fermes, ferment 听起来都一样），所以必须有主语来指明是谁。

但在 祈使句 (Imperative Sentence) 中，这一规则被指令性逻辑覆盖了：

- 唯一性： 命令的对象只能是“你 (tu)”、“我们 (nous)”或“你们 (vous)”。

- 情境化： 说话时，对象通常就在对面，语境已经完成了“主语显化”。

- 效率原则： 也就是我们反复提到的 语言经济原则 (Linguistic Economy)。为了让指令更具爆发力和冲击力，剥离主语是最有效的手段。

#### 命令式 + 宾语代词  vs  命令式 + 直接宾语名词

| 情况             | 说明                                                                               |
| -------------- | -------------------------------------------------------------------------------- |
| 动词后直接跟**名词宾语** | 不需要连字符。例：**Ferme la porte** → “Ferme”（命令式） + “la porte”（名词短语） → 语法上宾语明确，连字符没有必要。 |
| 动词后跟**代词宾语**   | 命令式后置宾语必须**连写用连字符**。例：**Apporte-moi** → “Apporte” + “moi”，中间用 **-**，是命令式语法要求。    |
| 多个代词宾语组合       | 按固定顺序用连字符连接。例：**Donne-le-moi** → 动词 + 宾格代词 + 间接宾语代词。                             |

### 感叹句
#### 元素结构顺序

| 顺序 | 成分名称 (Component) | 实例 (Example) | 逻辑功能 (Function) |
| :--- | :--- | :--- | :--- |
| **1** | **感叹引导词** | **Quel / Que / Comme** | **情感开关**：标记感叹的开始。 |
| **2** | **被感叹的核心** | **beau kilo / mûres** | **重点对象**：强调的品质或事物。 |
| **3** | **主语 + 动词** | **(ces tomates sont)** | **背景陈述**：交代感叹的主体。 |
| **4** | **标点符号** | **!** | **视觉强音**：书面语中的强制标识。 |
