# 主谓宾 Subject Verb Object
主系表描述“是什么/怎么样”，主谓宾描述“做了什么”。

## SVO vs. SLC

| 结构类型 | 英文概念 (English) | 中文概念 (Chinese) | 逻辑关系 | 功能说明 |
| --- | --- | --- | --- | --- |
| 主系表 | Subject-Linking Verb-Complement | 主系表 | 主语 = 表语 | 描述主语的状态、特征、本质或身份。 |
| 主谓宾 | Subject-Verb-Object | 主谓宾 | 主语 -> 宾语 | 描述主语发出的动作以及动作的承受者。 |
