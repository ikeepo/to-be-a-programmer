# 动词变位 Conjunction
## 动词变位有两大类

| 变位类别 | 包含形式 | 逻辑本质 (Logic) |
| :--- | :--- | :--- |
| 限定形式 (Modes personnels) | 现在时、未完成过去时、条件式等 | 必须随人称 (我、你、他) 发生变化。 |
| 非限定形式 (Modes impersonals) | 不定式、过去分词、现在分词 | 不随人称变化。它更像是一个“插件”，代表动作的状态。 |

#### 过去分词
##### 作用
过去分词是动词变位产生的一个零件。它自己不能独立表达时间（除非在省略句中），它必须配合助动词才能构成复合时态。

| 身份 | 所属层级 | 例子 | 作用 |
| :--- | :--- | :--- | :--- |
| 时态零件 | 时间层 | J'ai mangé. | 配合 avoir 构成复合过去时。 |
| 形容词 | 关系层 | Une porte fermée. | 描述名词的状态 (被关闭的门)。 |

##### 过去分词形变规则

| 动词组别 | 原形后缀 | 分词生成规则 | 例子 (阳单) | 阴性单数 (+e) | 阳性复数 (+s) | 阴性复数 (+es) | 逻辑注释 (Notes) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 第一组 | -er | 变为 -é | mangé | mangée | mangés | mangées | 绝大多数动词属于此类，发音始终为 /e/。 |
| 第二组 | -ir | 变为 -i | fini | finie | finis | finies | 词根通常稳定，发音始终为 /i/。 |
| 第三组 (U型) | -re/oir | 变为 -u | voulu | voulue | voulus | voulues | 常见于 vouloir, devoir, attendre 等动词。 |
| 第三组 (S型) | -re/oir | 变为 -is | pris | prise | pris | prises | 注意：阳单结尾已有 s，阳复不再加 s。 |
| 第三组 (T型) | -re/ir | 变为 -it/ert | fait | faite | faits | faites | 常见于 faire, écrire, offrir 等动词。 |

## 动词变位有五个维度

| 法语术语 | 中文 | 英文 | 维度功能 |
| :--- | :--- | :--- | :--- |
| **La Personne & Le Nombre** | 人称与数 | Person & Number | 确定执行动作的主体是谁（单/复数）。 |
| **Le Temps** | 时态 | Tense | 确定动作发生的时间点（过去/现在/将来）。 |
| **Le Mode** | 语式 | Mood | 体现说话人的语气（事实/假设/命令/主观）。 |
| **La Voix** | 语态 | Voice | 区分动作是主动发出还是被动接受。 |

## Relationship
### 主语显化
动词必须有明确的主语代词（如 Il fait 中的 Il），因为动词变位本身往往不足以在听觉上区分人称。


## Rules
### 法语动词分类

| 类别 | 结尾 (Ending) | 特点 (Characteristics) | 例子 (Example) |
| :--- | :--- | :--- | :--- |
| **第一组** | **-er** | **绝对规则**，占法语动词 90% 以上。 | *Parler, Fermer, Apporter* |
| **第二组** | **-ir** | **相对规则**，变位时复数人称会生出 `iss`。 | *Finir, Choisir* |
| **第三组** | **-re / -oir / 不规则 -ir** | **不规则/大杂烩**，变位多变，但最为常用。 | *Être, Avoir, Faire, Aller* |

### 基于人称的变位
#### 动词变位规则: `er`动词 
先去掉`er`然后按照如下规则：

| 人称代词 (Subject) | 词尾 (Ending) | 示例：Parler (Speak) |
| :--- | :--- | :--- |
| {{fr('Je')}} | **-e** | Je parle |
| {{fr('Tu')}} | **-es** | Tu parles |
| {{fr('Il')}} / {{fr('Elle')}} / {{fr('On')}} | **-e** | Il parle |
| {{fr('Nous')}} | **-ons** | Nous parlons |
| {{fr('Vous')}} | **-ez** | Vous parlez |
| {{fr('Ils')}} / {{fr('Elles')}} | **-ent** | Ils parlent |

#### 动词变位规则: 不规则动词
##### `aller`

| 人称 (Person) | 变位 (Conjugation) | **IPA 发音** | 助记/规律 (Logic) |
| :--- | :--- | :--- | :--- |
| **Je** | **{{fr('vais')}}** | **/vɛ/** | 词尾 s 不发音，开口音 /ɛ/。 |
| **Tu** | **{{fr('vas')}}** | **/va/** | 词尾 s 不发音，纯正 /a/ 音。 |
| **Il/Elle/On** | **{{fr('va')}}** | **/va/** | 形式最简，发音与 Tu 一致。 |
| **Nous** | **{{fr('allons')}}** | **/a.lɔ̃/** | 词根回归，注意鼻化音 /ɔ̃/。 |
| **Vous** | **{{fr('allez')}}** | **/a.le/** | 词根回归，词尾 -ez 发 /e/ 音。 |
| **Ils/Elles** | **{{fr('vont')}}** | **/vɔ̃/** | 特殊形态，类似 *ils sont* 的发音逻辑。 |

| 人称 (Person) | 变位 (Conjugation) | **IPA 发音** | 助记/规律 |
| :--- | :--- | :--- | :--- |

| **Ils/Elles** | **{{fr('sont')}}** | **/sɔ̃/** | 鼻化音 /ɔ̃/ |

##### `avoir`


- 直陈式现在时

| 人称 (Person) | 变位 (Conjugation) | **IPA 发音** | 助记/规律 |
| :--- | :--- | :--- | :--- |
| **J'** | **{{fr('ai')}}** | **/e/** | Je + ai 省音缩合 |
| **Tu** | **{{fr('as')}}** | **/a/** | 词尾 s 沉默 |
| **Il/Elle/On** | **{{fr('a')}}** | **/a/** | 最简形态 |
| **Nous** | **{{fr('avons')}}** | **/a.vɔ̃/** | av- 词根 + 鼻化音 |
| **Vous** | **{{fr('avez')}}** | **/a.ve/** | av- 词根 + /e/ |
| **Ils/Elles** | **{{fr('ont')}}** | **/ɔ̃/** | 仅发鼻化音，与 sont 区分 |

- 条件式现在时

##### `dire`

| 时态 (Temps) | Je / J' | Tu | Il / Elle / On | Nous | Vous | Ils / Elles |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **Présent** (现在时) | dis | dis | dit | disons | **dites** | disent |
| **Imparfait** (未完成过去时) | disais | disais | disait | disions | disiez | disaient |
| **Futur Simple** (简单将来时) | dirai | diras | dira | dirons | direz | diront |

| **Subjonctif Présent** (虚拟式) | dise | dises | dise | disions | disiez | disent |
| **Impératif** (命令式) | - | dis | - | disons | **dites** | - |

### 基于时态的变位
#### 条件式现在时 

| 维度 | 分析内容 | 逻辑本质 (Logic) |
| :--- | :--- | :--- |
| 词法构成 | 简单将来时的词根 + 未完成过去时的词尾 | 这种组合代表了“指向未来”但“带有过去的不确定感”。 |
| 语义功能 | 表达礼貌、委婉或假设 | 相比直陈式的 veut (他想要)，voudrait 听起来是“他想... (如果可以的话)”。 |

#### 复合过去时

## 所有人称、所有时态下变位规则
