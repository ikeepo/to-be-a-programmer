# 主系表 SLP
Subject Linking Verb Predictive.

这个结构是一个等式： `A(主语) = B(表语)，系动词就是那个“等号”`

| 成分 | 英文概念 (English) | 中文概念 (Chinese) | 功能说明 |
| --- | --- | --- | --- |
| 主语 | Subject | 主语 | 动作的发起者或被描述的对象 |
| 系动词 | Linking Verb | 系动词 | 连接主语和表语，表达状态或身份 |
| 表语 | Complement | 表语 | 赋予主语具体的特征、身份或性质 |

## 表语构成
表语需要跟主语保持性数一致。

| 词性 | 英文概念 (English) | 中文概念 (Chinese) | 源码示例 |
| --- | --- | --- | --- |
| 形容词 | Adjective | 形容词 | Elle est contente. (她很高兴) |
| 名词 | Noun | 名词 | Mon père est médecin. (我父亲是医生) |
| 代词 | Pronoun | 代词 | Ce livre est le mien. (这本书是我的) |
| 动词不定式 | Infinitive | 动词原形 | Vouloir, c'est pouvoir. (有志者事竟成) |
| 介词短语 | Prepositional Phrase | 介词短语 | Il est en colère. (他在生气) |
