# 时态 Tense

## 时态类型

| 优先级 | 时态名称 | 身份标识 (Identifier) | 核心逻辑 | 经典实例 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **现在时** | 词尾变位 (-e, -s, -t) | 现状、真理、习惯 | *Je mange.* |
| **2** | **复合过去时** | **avoir/être + 过去分词** | 已完成的点状动作 | *J'ai mangé.* |
| **3** | **最近将来时** | **aller + 动词原形** | 即将发生的计划 | *Je vais manger.* |
| **4** | **未完成过去时** | 词缀 **-ais, -ais, -ait** | 过去的背景/状态 | *Je mangeais.* |
| **5** | **简单将来时** | **r + -ai, -as, -a** | 预测或遥远的未来 | *Je mangerai.* |
| **6** | **最近过去时** | **venir de + 动词原形** | 刚刚发生的动作 | *Je viens de manger.* |
| **7** | **愈过去时** | **avais/étais + 过去分词** | 过去的过去 | *J'avais mangé.* |

### 二：复合过去时
它展示了法语如何处理“已完成的动作”。
复合时态是专指动词由两部分构成的时态。

`助动词 + 过去分词`


### 三：最近将来时 futur proche
表示即将发生或意图明确的动作。助动词只有`ALLER`这一个。

最近将来时跟`aller`深度绑定，[`aller(直陈式现在时变位) + 动词原形`](./conjugation.md#所有不规则动词的所有变位形式)

aller + 名词/副词 → 现在时意思：“去某地”

aller + 不定式动词 → 近未来意思：“将要做某事”

### 简单未来时 futur simple

| 特征   | 近未来 (futur proche)               | 简单未来 (futur simple)             |
| ---- | -------------------------------- | ------------------------------- |
| 构成   | aller (现在时) + 不定式                | 动词原型 + 未来词尾                     |
| 时态感  | 临近将来，马上发生                        | 未来某时，可近可远                       |
| 使用场景 | 口语、生活场景、计划、预测                    | 书面语、正式场景、预测、预言、承诺               |
| 例句   | Je **vais manger**. → 我要吃（马上/今天） | Je **mangerai** demain. → 明天我会吃 |
| 语气   | 口语、自然                            | 正式、中性                           |


口语中，母语者几乎总用近未来，听起来更自然

### 条件式现在时 
并非复合时态，是简单时态，单一动词形式，任何动词都可以实现条件式现在时。
将“欲望”或“动作”包裹在一层礼貌或假设的缓冲垫中。
产生[动词变位](../morphology/conjugation.md#条件式现在时). 
类似英语的`would`.


#### 时态结构特征
- 动词变位：`条件式 = 未来时词干 + 未完成过去时词尾` 

- 常与固定短语 `d’accord` 和 `pour` + 不定式 连用形成完整礼貌请求

#### 如何识别？

| 识别维度 | 书面识别 (Literacy) | 口语识别 (Orality/文盲模式) |
| :--- | :--- | :--- |
| 核心工具 | 眼睛 (拼写) | 耳朵 (音位) |
| 关键线索 | 字母 r + ait/ais/aient | 颤音 /r/ + 开口音 /ɛ/ |
| 逻辑依赖 | 词法层公式 | 语流层节奏与社交语境 |
| 结论 | 通过规则推导 | 通过生存本能与语感内化 |

### 条件式过去时
```shell
助动词avoir 或 être 跟随主语性数一致
+ 
主要动词的过去分词
```
主要动词的过去分词是否变化取决于助动词，助动词是`avoir`则不需要与主语保持性数一致；如果助动词是`être` 则主要动词要与主语保持性数一致。
