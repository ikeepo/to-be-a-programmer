# 动词 Verbe

## 助动词 Auxiliaire
动词中失去独立语义，仅提供辅助的部分，一种功能性词汇。
### 应用场景

| 场景类别 | 核心功能 | 常用助动词 | 典型例子 | 逻辑本质 |
| :--- | :--- | :--- | :--- | :--- |
| 复合时态 | 构建时间深度 | avoir / être | J'ai fini / Je suis allé | 区分“正在做”与“已做完”。 |
| 被动语态 | 转换动作视角 | être | La porte est fermée | 动作的承受者变成了主语。 |
| 代动词 | 处理自发动作 | être | Je me suis lavé | 动作回到了主语自己身上。 |
