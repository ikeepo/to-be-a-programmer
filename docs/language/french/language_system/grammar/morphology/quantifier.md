# 量词 Quantifiers

## 冠词des vs 量词quelques
冠词des用于用于不可数名词或抽象物质；

量词quelques后面必须跟名词复数，不可直接与不可数名词连用.

#### 举例
`tu serais d’accord pour faire quelques brasses`

每个蛙泳动作可以计数。

## 量词类型

### `量词+de+名词`
前面不再需要冠词；
冠词也可以修饰形容词，此时没有`de`;
### `固定表达`
量词本身带有冠词，冠词不是用于修饰名词，而是属于量词的一部分；

# `de + 形容词 + 名词复数`

`de nombreuses discussions`

形容词前置，`des`改为`de`,这里的形容词只有两类：

- 数量/程度类形容词

- 评价/性质类形容词
