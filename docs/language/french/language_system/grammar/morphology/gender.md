# 性Gender
## Abstract
归属于哪边并没有明确的规则，更多是历史传承。

对于法语学习者来说，性别不是用来“理解”的，而是作为名词的原生属性（像 DNA 一样）一起记忆的。


## Rules
### e结尾的很多是阴性
在拉丁语中，以 -a 结尾的名词绝大多数都是阴性。拉丁语结尾的 -a 在法语中演变成了哑音 -e。
### 如何判断阴性阳性
#### 方式一：有生物性别的依据生物性别
#### 方式二： 行业与学科逻辑 (Category Logic)
#### 方式三： 形态后缀
##### 方式3.1：结尾e ·
非e结尾的归到阳性，初步推断（Educated Guess）可以这样判断70-80%；
##### 方式3.2：结尾词缀
真正的阳性信号 (Masculine Suffixes):
-age: Le voyage, le garage (例外：la plage, la cage)

-eau: Le bureau, le bateau

-isme: Le réalisme, le cyclisme

-ment: Le gouvernement, le moment

-phone / -scope: Le téléphone, le microscope

真正的阴性信号 (Feminine Suffixes):
-tion / -sion: La nation, la décision (极高概率)

-té: La liberté, la cité (通常指抽象名词)

-ette: La bicyclette, la baguette

-ance / -ence: La chance, la différence

- 通常为阳性： 季节、月份、星期、金属、树木、语言

- 通常为阴性： 所有的学科、水果名（大部分）

## 性别转变
大部分是固定的，职业等可能改变，这是Agreement layer在做的事情。` Un avocat (男律师) $\rightarrow$ Une avocate (女律师)。`
### 原型与变位
原型 (Base Form): 在法语词典和传统语法架构中，阳性单数形式 (Masculin Singulier) 被视为“原型”或“默认形式”。

衍生 (Derivative): 阴性形式 (Féminin) 是在阳性的基础上，通过添加或修改后缀生成的。

### 变位规则
- 基础规则：直接加 -e

- 结尾替换规则 (Suffix Transformation)

如果阳性词尾已经是某种特定形式，则需要进行“手术式”替换：-er $\rightarrow$ -ère: Un boulanger $\rightarrow$ Une boulangère (面包师)-eur $\rightarrow$ -euse: Un danseur $\rightarrow$ Une danseuse (舞者)-teur $\rightarrow$ -trice: Un acteur $\rightarrow$ Une actrice (演员)-en/on $\rightarrow$ -enne/onne: Un pharmacien $\rightarrow$ Une pharmacienne (药剂师)

- 中性形式 (Epicene Nouns)

如果阳性原型本来就以 -e 结尾，那么阴阳性长得一模一样，只靠冠词区分。
## Relationship
### 协调层 (Agreement)
名词锁定了阴性，它就会在协调层 (Agreement) 引发连环反应。
### 语序 (Syntaxe)
性的存在增加了法语的冗余度 (Redundancy)。在语流（Speech Stream）中，如果你听到 la...，你已经知道了后面的名词是阴性。如果后面接了多个形容词，每一个形容词的阴性后缀都在反复确认这个“性”，这能有效防止听者理解偏差。
### 语体 (Registre):
在所有语体中，名词的性别是不可动摇的。说错性别（把 La porte 说成 Le porte）是极高频率的错误，在 TCF 考试中会被视为基础架构不稳。
