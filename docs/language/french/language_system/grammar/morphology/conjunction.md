# 连词 Conjunction
不可变词类；

我们将其分为 Coordinating Conjunctions（并列连词）与 Subordinating Conjunctions（从属连词）。

## 结构

| 类别 | 逻辑关系 | 常用连词 | IPA (音标) | 逻辑本质 |
| :--- | :--- | :--- | :--- | :--- |
| 并列 | 并列/增加 | {{fr('et')}} (和) | /e/ | 简单加法：A + B。 |
| 并列 | 选择 | {{fr('ou')}} (或者) | /u/ | 二选一：A 或 B。 |
| 并列 | 转折 | {{fr('mais')}} (但是) | /mɛ/ | 逻辑冲突：A，但 B。 |
| 并列 | 因果 | {{fr('car')}} (因为) | /kaʁ/ | 逻辑推导：因为 A。 |
| 并列 | 因果 | {{fr('donc')}} (所以) | /dɔ̃k/ | 逻辑推导：所以 B。 |
| 从属 | 引导/声明 | {{fr('que')}} (那/这...) | /kə/ | 连接主句动作与具体内容。 |
| 从属 | 时间 | {{fr('quand')}} (当...时) | /kɑ̃/ | 建立时间轴。 |
| 从属 | 原因 |  {{fr('parce que')}} (因为) | /paʁ.sə.kə/ | 建立因果链条。 |
