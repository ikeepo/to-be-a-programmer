## 实词 vs 虚词 

| 类别 | 包含词类 | 逻辑本质 | 助动词的归属 |
| :--- | :--- | :--- | :--- |
| 实词 (Content Words) | 名词、形容词、普通动词、副词 | 携带具体的、实在的意义。 | 普通动词（如 manger）属于此类。 |
| 虚词/功能词 (Function Words) | 介词、连词、冠词、助动词 | 意义抽象，主要负责搭建语法结构。 | 助动词在这个维度上和介词、冠词是同伙。 |
