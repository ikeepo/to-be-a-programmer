# 限定词 Determiner
法语名词几乎必须有限定词；限定词是名词短语的核心结构。

句型结构主要由虚词构建，而限定词是法语最核心的虚词系统之一。

| 类型    | 法语                                     | 英文                  |
| ----- | -------------------------------------- | ------------------- |
| 定冠词   | le / la / les                          | the                 |
| 不定冠词  | un / une / des                         | a / some            |
| 部分冠词  | du / de la / des                       | some                |
| 指示限定词 | ce / cet / cette / ces                 | this / that / these |
| 所有限定词 | mon / ton / son / notre / votre / leur | my / your / his     |
| 疑问限定词 | quel / quelle / quels / quelles        | which               |
| 数词    | un / deux / trois                      | one / two           |
| 不定限定词 | plusieurs / certains / quelques        | several / some      |
| 否定限定词 | aucun                                  | none                |

## 举例 

| 类型   | 法语               | 英文            |
| ---- | ---------------- | ------------- |
| 定冠词  | le livre         | the book      |
| 不定冠词 | un livre         | a book        |
| 指示   | ce livre         | this book     |
| 所有   | mon livre        | my book       |
| 数量   | deux livres      | two books     |
| 不定   | plusieurs livres | several books |
| 疑问   | quel livre ?     | which book    |
