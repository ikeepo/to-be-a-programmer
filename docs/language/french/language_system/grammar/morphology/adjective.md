# 形容词 Adjective

### 法语物主形容词表 (Possessive Adjectives)
对于“物主形容词”本身，复数确实不分阴阳；但对于“普通形容词”，复数依然要看阴阳。

| 人称 | 阳性单数 / IPA | 阴性单数 / IPA | 复数 (不论阴阳) / IPA |
| :--- | :--- | :--- | :--- |
| **Je** (我的) | **{{fr('mon')}}** /mɔ̃/ | **{{fr('ma')}}** /ma/ | **{{fr('mes')}}** /me/ |
| **Tu** (你的) | **{{fr('ton')}}** /tɔ̃/ | **{{fr('ta')}}** /ta/ | **{{fr('tes')}}** /te/ |
| **Il/Elle** (他/她的) | **{{fr('son')}}** /sɔ̃/ | **{{fr('sa')}}** /sa/ | **{{fr('ses')}}** /se/ |
| **Nous** (我们的) | **{{fr('notre')}}** /ˈnɔtʁ/ | **{{fr('notre')}}** /ˈnɔtʁ/ | **{{fr('nos')}}** /no/ |
| **Vous** (你们/您的) | **{{fr('votre')}}** /ˈvɔtʁ/ | **{{fr('votre')}}** /ˈvɔtʁ/ | **{{fr('vos')}}** /vo/ |
| **Ils/Elles** (他们的) | **{{fr('leur')}}** /lœʁ/ | **{{fr('leur')}}** /lœʁ/ | **{{fr('leurs')}}** /lœʁ/ |


**⚠️ 特殊规则 (Special Rule)：**
如果阴性单数名词以**元音**或**哑音 h** 开头，为了语音顺滑，必须使用阳性形式。
* *Ma amie* (❌) -> ***Mon** amie* (✅)

### 指示形容词 
作用更像冠词，必须放在名词前面，与名词保持性数一致。

称其为指示的意思是说，它不描述名词的性质，而是限定名词的范围。

指示形容词只有`ce`这一组：

| 形式    | 性数    | 例子           | 中文   |
| ----- | ----- | ------------ | ---- |
| ce    | 阳性单数  | ce livre     | 这本书  |
| cet   | 阳性元音前 | cet homme    | 这个男人 |
| cette | 阴性单数  | cette maison | 这栋房子 |
| ces   | 复数    | ces enfants  | 这些孩子 |


# 形容词位置
一般形容词放在名词的后面，但是有一些列外：BAGS


| 类别           | 含义               |
| ------------ | ---------------- |
| B = Beauty   | 美丽（beau, joli）   |
| A = Age      | 年龄（jeune, vieux） |
| G = Goodness | 好坏（bon, mauvais） |
| S = Size     | 大小（grand, petit） |
