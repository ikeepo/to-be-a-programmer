# 法语形变的bug
我发现一个发音现象，虽然形变很多，但是很多时候，一些虚词、功能词的形变后发音是相同的，这就导致虽然大脑无法第一时间理解时态、性数一致等规则，但是并不影响听者的理解。

法语里大量的：时态变化/性数变化/词形变化，在发音上高度“塌缩”（phonetic collapse）。

法语是一个“拼写复杂，但听觉极简”的语言，语法信息大量存在于“写”里，而不是“听”里

人类语言本就是先有口语，而书写系统是很久之后的存在。

这意味着，就听力而言，甚至可以：

- 不完全掌握变位规则

- 不精确区分性数一致

- 不完全理解时态细节

依然可以通过
