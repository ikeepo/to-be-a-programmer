# 听力并不是听词的完整读音来调用意思
因为音节组打乱了词对音节的边界界定，所以是通过场景（收敛特定词范围）+特定词的关键音节，预测出具体的词。这是一个认知预测算法，而不是单词读音调用算法。预测算法的输入是：场景+特定音节，预测出意思，再通过其他特定音节预测出的词进行验证。

听力并不是“等到完整词出现再识别”，而是持续的增量式识别（incremental processing）.

# 听力识别模式

# 记忆模型
脑细胞并不会伴随新陈代谢更新，基本是伴随人体永久的。

记忆是大脑里长出新的微观结构，持续刺激，这种微观结构（dendritic spine）就变大、变多、形状改变。

短期记忆（小时）只改变审计递质释放、点活动模式，不涉及结构改变。

长期记忆需要蛋白质合成、突触结构改变。

记忆并不存储在细胞内，而是突触（神经元之间的连接）在结构和化学状态上的长期改变。

一个神经元可以连接上千个其他神经元，`记忆=哪些神经元连在一起+连接又多强`。

白天记忆存储在海马体（临时缓存），晚上海马体replay记忆，写入神经连接。

同时，REM睡眠（做梦阶段），信息会介入已有知识体系。

写入过程有一个过滤器：情绪（杏仁核），冥想正是在这个环节起作用。

冥想会降低过度情绪反应，减少“错误强化”。

连续刷（短时间），同一突触刚刚激活，没有“遗忘压力”，大脑会判定为低价值信息。

- 视觉/听觉输入（感知区）

- 意力锁定（前额叶）

- 意义绑定（语言区 + 海马体）

- 价值判断（情绪系统）

专注可以使多个功能区联动，构建记忆网络。
