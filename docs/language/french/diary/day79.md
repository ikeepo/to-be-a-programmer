# 形式主语

`C’est [cela] [la profondeur de la culture]`

| 结构                  | 本质功能              |
| ------------------- | ----------------- |
| **C’est + 名词/指代**   | 指认 / 定义 / 强调“是什么” |
| **Il est + 形容词/身份** | 描述状态 / 性质         |


# `a vs dans`

- à = 点（机构/地点）

- dans = 面（空间/区域）

| 中文  | 法语               | 类型     |
| --- | ---------------- | ------ |
| 在学校 | à l’école        | 机构/功能点 |
| 在医院 | à l’hôpital      | 机构     |
| 在家  | à la maison      | 固定表达   |
| 在国家 | dans un pays     | 空间区域   |
| 在城市 | dans une ville   | 区域     |
| 在房间 | dans une chambre | 空间     |
