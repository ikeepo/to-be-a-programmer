# 完成Task3的第一遍背诵
现在对于task3的内容，看几遍，背诵三五次就能记住一个句子，但是让我自己说，还是远远不灵活。

### 能够识别、复现，不能生成
语言能力的三个阶段：

- 识别

- 复现

- 生成

```shell
# 复现路径
文本 -> 记忆 -> 输出

# 生成路径
想法 -> 结构选择 -> 词汇选择 -> 输出
```

背诵和自己说，完全是两条不同的路径。同为训练基地神经节的自动化，这两者训练的完全是不同的自动化路径，并不具有明确的相关性。


而自动化路径的不同，根本上是**检索线索**的不同，**背诵**是一种序列记忆sequence recall，检索线索是临时构建的缓存，临时调用，过了几天，这个临时构建的缓存已经被冲刷覆盖，没有可供调用的线索，也就无法背诵成功。

而**生成路径**是一种基于概念的检索，基于主体概念只能得到一系列的词汇，这些词汇是零散的。而人类语言的表达收敛性决定了语言的表达逻辑只有固定的几种。进一步对于口语考试而言，主要是：问题（task2）、观点（T3）、举例（T3），这时候就是模版在起作用，就是说，模版是一种对于*表达功能*的线索。

- 用*模板*作为*表达结构*的线索【生成】

- 用*概念*作为*相关主体词汇*的线索【生成】

- 用*文本构成的临时缓存*作为*序列记忆*的线索【背诵】

# 背诵的价值是建立数据库
背诵这个过程分成两个环节：顺序输出成功+建立语言数据库。

前者是线索--输出路径，因为线索是临时缓存，没有太大的意义；

后者是留存的零碎的表达能力，是口语表达的底层检索原子数据，是背诵过程留存的有价值工作量。

从这个角度，背诵，并不需要后续能够持续的顺序输出，但是要求背诵的素材本身经过很好的整理，具有足够的重复度。

这就像给小孩子看的动画片，没必要像葫芦娃一样就7集反复播放，可以像papa一样，几百集但是用的词汇反复重复，但是构建起这些词汇的。
