# 语音块与语义块并不准确对应
语音块以流畅为主，这就导致听力训练在累计语音块、语义块的同时，需要理解语义块跨音节的规律。

法语语义识别类似中文，高度依赖上下文，也就是场景识别。
# 法语听力合理映射流程：听力识别算法
```shell
声音流输入
→ 抓锚点
→ 激活高频模板
→ 用语法接口切分
 功能词重要--映射到高频句型
→ 形成候选语义块
→ 识别场景脚本
→ 预测后续逻辑
→ 用上下文验证/修正
```
# 听力识别算法分层
```shell
第一层：锚点识别

听出高频词、时间词、态度词、句型接口。

第二层：模板识别

整块识别：

ça te dit de...

tu veux...

j’ai envie de...

il faut...

je suis fatigué

第三层：场景识别

这是邀约、拒绝、解释、问路、购物、电话、约会、工作沟通中的哪一类。

第四层：预测与验证

根据脚本预判下一句，再用实际听到的内容修正。
```
