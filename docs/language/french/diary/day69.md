# 语音及意义识别范式

虽然有同音、同义、同音异意等各种语言上的tricky，但是当一个词跟场景匹配后，在同等场景下合适的替换并不会很多；**语言是收敛的，不是无限发散的**。这就是说，语言能力不是词汇量问题。

所以发音差异、小错误并不影响交流，但这要求大脑已经对这种场景及这些话语建立了数据库，并构建了意义到语言的快速检索算法。

而这种检索算法及数据库的建立，需要几百次的场景及音频对大脑的暴露.
