# 大脑训练模型

| 层级            | 类比                 | 时间   |
| ------------- | ------------------ | ---- |
| cache         | working memory     | 秒–分钟 |
| batch replay  | hippocampus        | 小时   |
| weight update | cortex integration | 天    |

学习一次 = 一次 batch input

## 定义一次好输入
一次输入（比如看剧/学习/阅读）在大脑里本质是：**外部信息 → 内部建模 → 压缩(结构而非instance) → 写入长期记忆**。

存在一个效率及带宽配合问题。

单次输入：

- 45–90分钟是最优区间（工作记忆容量开始饱和、DMN开始抢占资源）

- 结构完整度（如果未闭环，DMN会自动补全，也就是错误

- 从instance中总结出结构，为DMN的自监督预测收敛过程人工标注约束信号
