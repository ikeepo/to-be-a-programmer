# 游戏设计

将这个功能收集的环节做成一个编程项目，最好是某种游戏形式，比如，有一个小人，当我获得某种功能，我就可以解锁一步移动，或者扩展房屋，比如，原本我是在卧室，只有当能掌握厨房用的句子后才能进入厨房场景。

空间化语言进度系统（Spatial Learning Progression Game）

本质是：

语言能力 = 功能权限（ability unlock）
学习过程 = 地图探索（map unlocking）
记忆系统 = 空间记忆（spatial memory）
