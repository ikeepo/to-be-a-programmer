# CEFR vs. CLB 

| 维度   | CEFR（A/B/C） | CLB        |
| ---- | ----------- | ---------- |
| 核心   | 语言结构复杂度     | 实际沟通任务     |
| 关注点  | 句子有多复杂      | 能不能把事办成    |
| 测试方式 | 语法+表达       | 场景+任务      |
| 本质   | **语言能力模型**  | **功能能力模型** |

# 口语表达的物理结构拆解
`最底层 = 功能 + 句型 + 词汇`  

对于听力，还需要一个纬度：`场景识别`

```shell
表达的底层是：功能 + 句型 + 词汇
理解的底层是：场景（功能） + 句型 + 词汇

场景 = 功能 + 一些额外信息 Context
```

### CLB5不是语言水平，而是“反应速度 + 功能表达能力”。
对于CLB考试，这里还有一个纬度需要训练：调用自动化。这个通过高频暴露来实现。

### 词汇
不走“背单词”路线，走“功能绑定词汇”路线。

### 高频重复建立自动化映射
实现稳定反射，一个表达需要100-300次有意义的重复；

### “功能”掌握标准
任意功能，可以连续输出5句，不停留超过1s，能换3个场景；

### 训练素材
以为考试题目准备答案为主，可以用课本表达进行验证或者泛化。
