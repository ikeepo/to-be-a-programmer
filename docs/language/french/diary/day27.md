## 法语等级考试各个层级口语要求


| 语法维度     | CLB5                                                                                                                                                                                   | CLB6                                                                                                             | CLB7                                                                 | CLB8                                                                     |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| 核心语法知识考查 | 基本动词变位（直陈式现在时、将来时、条件式现在）；常用不规则动词：être, avoir, faire, aller, pouvoir, vouloir, savoir, devoir                                                                                           | 增加基础过去时：passé composé；不规则动词变位更熟练                                                                                 | 增加 imparfait, plus-que-parfait 基础使用                                  | 增加 Subjonctif présent 初步使用，虚拟语气表达愿望、必要性、怀疑                               |
| 时态/语气    | Présent, Futur simple, Conditionnel présent                                                                                                                                            | Passé composé（基础）                                                                                                | Imparfait, Plus-que-parfait（基础）                                      | Subjonctif présent                                                       |
| 句型结构     | 简单句（主语 + 谓语 + 宾语/补语）；少量简单复合句：si, que, quand, parce que, après, pour, lorsque, dès que, tandis que, comme                                                                               | 简单复合句增多；连词：parce que, après que, lorsque, dès que, tandis que, comme, puisque, bien que, alors que；表达因果、时间、目的、让步 | 复合句使用广泛（因果、条件、时间、目的、让步、比较、原因）；连词增量：malgré que, tandis que, alors que | 复合句更复杂多样；连词增量：pour que, afin que, malgré que, bien que, quoique          |
| 代词/冠词    | 主格代词：je, tu, il/elle/on, nous, vous, ils/elles；宾格代词：me, te, le, la, l’, nous, vous, les；间接宾语：lui, leur；y / en（基础）；冠词：un, une, le, la, l’, des；量词：quelques, beaucoup, plusieurs, un peu | 代词使用增量：y, en, me, te, se 的熟练运用                                                                                   | 代词/冠词增量：代词短语使用准确，包括 lui/leur/y/en 复合搭配；量词使用灵活                        | 代词/冠词增量：代词短语熟练使用，包括 y/en 与宾语结合                                           |
| 条件/从句    | 条件句简单：si + 直陈式；宾语从句：que + 直陈式；简单目的从句：pour + 不定式                                                                                                                                        | 条件句增量：si + futur / conditionnel；目的从句增量：pour que, afin que；宾语从句可使用将来时                                             | 条件句复杂化：si + imparfait / conditionnel；从句增量：让步从句、比较从句、假设表达             | 条件/从句增量：si + conditionnel + subjonctif；虚拟语气从句（souhait, doute, nécessité） |
| 特点 / 说明  | 能表达日常交流、建议、意图、喜好；语法错误偶尔出现不影响理解                                                                                                                                                         | 句子更连贯，自然表达日常交流、建议、计划；语法错误偶尔出现但不影响意思                                                                              | 能完整描述过去/未来事件；逻辑清楚；语法错误少；复合句自然                                        | 表达流畅、连贯、逻辑清晰；可处理复杂话题（工作、社会、抽象）；语法错误很少                                    |
