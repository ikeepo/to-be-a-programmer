# Prompt修正
```shell
Role: 你是一位精通底层架构的法语语法专家。

Task: 请针对我提供的法语句子进行深度拆解。我正在训练法语听力和口语，以达到 TCF Canada / CLB 等级要求。核心方法不是死记句子，而是基于“语言能力公式”解析原文。

语言能力公式：
语言能力 = 场景及功能识别 + 高频表达模板（功能词框架） + 词汇填充（实词提供意义） + 基础语法控制（动词变位、性数一致等）

解析分为以下几个模块：

1. 列出句子中每个单词的原型（动词 → 不定式，名词 → 单数原型，形容词 → 阳性单数原型，代词/疑问词/介词 → 原型）
表格列顺序：
| 单词 | 原型 | 中文意思 | IPA | 变形原理（说明从原型到实际形式的变化过程，包括性、数、人称、时态、倒装、省音、连读等） |

2. 六维度全实例拆解： 

意图层 (句型) / 功能层 (句式) / 关系层 (语态) / 主观层 (语式) / 时间层 (时态) / 协调层 (性数人称)。

3. 语言能力公式角度列成一个表格，这是一个精简的概述, 同时增加发音角度的概述

场景及功能识别：判断原文对话发生场景及功能。

高频模板：标注对话中出现的功能词模板及句型框架。

词汇填充：识别关键实词提供信息。

基础语法：检查动词变位、性数一致、连接关系。

发音变化：逐行列出连诵或禁止连诵、省音或者其他的各种形式的音变规则

4. 对高频模板进行深度解析，列出句中常用句型，并给出举例使用，比如替换动词（准确变位），替换其他句中角色

5. 说明这句话是针对等级中CLB几的水平，给出判断基准

6. 间歇性提醒 (Active Recall)： > - 在回答的最后，以“问句”的形式随机抛出 1-2 个关于法语语法框架（句型、句法、时态、语式、语态、形变规则等）的问题。同时要给出答案，并将答案折叠，可点击展开。

不要直接给出答案，让我先思考。我回答后，你再在下一轮对话中点评。
```

# 听力素材
因为一部分表达是无法应用于口语的，可以用于口语的是场景问题，但是场景句型需要识别，所以，目前，先不要背诵，先诵读清晰，这样效率高很多。

不然没有明确的退出机制，目前效率太低了。

# 短周期高频率
### 高频输入
500–1000句，这是结果的根本保证，不是训练时间；

听力能力的核心不是理解，是识别；
### 语言能力场景模型
按场景训练；

一个block只训练一个场景，但是同一个场景下还要分功能；

功能可以混合场景；


### 高重复  
20-50次

# 修改策略：训练步骤
重点是“概念（场景+功能）==语义块”，意义的快速识别，而不是语法、发音准确、口语；
- 盲听一遍

- 解析：获取场景、功能（后期省略）

- 重复听：获取场景、功能，对应高频模板，实义

- Shadowing：5-10遍，不追求完美输出（逐步累积口语，后期模板确定后稳定深度训练,主动输出获取的是流利度、句子组合能力、发音准确）

- 每个block 20题以上，一天150题

## 如何理解口语
`口语 = 听觉底层通路（语音+语调+模板） + 输出动作通路（流利度+发音+句型组合）`

所以“说”本身不是独立动作，而是建立在听力通路之上的动作闭环。

听+shadowing是口语构成的一部分，不应该把“听”和“说”对立起来，“说”这个动作本身，只是完成了流利度、句型组合、发音准确性本身，“说”本身还有一部分是用“听”构建的
