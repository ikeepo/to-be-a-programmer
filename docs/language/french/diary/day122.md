## 2. Task3化
### 2.1 用固定框架重新表达
这里不是复述原来题目，而是从自己整理的Task3中调取已经有的观点、素材借助这个题目的主题表述出来。

如果卡壳，不要停留太久，用中文替代表达，顺利完成。
```shell
Je pense que... ==> 观点 

Parce que...    ==> 解释

Par exemple...  ==> 例证

Donc...         ==> 结论
```

每一步中，表达方式有不同的选择，不是固定的。有基于场景及上下文更合适的表达方式。
