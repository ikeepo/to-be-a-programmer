# 听力的两种理解方式
- **等理解完成**{: #waiting_understand}

听到单词--翻译--拼意思--完成理解；被动等待所有意思出来

- **让理解发生**{: #letting_understand}

有限信息--直接生成意思；主动基于不完整信息预测意义，也就是缺乏所谓的**自动化**。

所谓自动化，就是语料被自动绑定到某个意思，但是现在没能绑定，所以训练的时候，要强行自动绑定一个意思，允许是错的，后续再纠偏。

也就是之前谈到的，语言不是对语料的直接感知，而是基于语料和场景的预测。
