# `intéressant` vs `reconnaître`
法语多是2，3音节词，这种4音节词相对少，但同样是4音节词，reconnaître明显比intéressant难读。

难读是因为不同音节发生器官变化幅度大。
# 法语发音困难类型总结
法语发音困难是声音在器官中位移大小决定，因此不是单个词的问题，是这些位置变化的问题。

位置变化是相对固定的，

# 母语者读的更顺利是因为肌肉更适应么？
NO，肌肉更顺畅是结果，不是原因。根本原因是母语者语音反射路径与外语学习者不同。

任何语言母语者中都有文盲，语言能力不等于识字能力。

母语者：意思--语音chunk

外语学习者：意思--单词--音节--语法应用--语音

这个映射过程是非常长的，效率自然低。
# 如何获取母语者的语言能力

强制输出训练，不允许停顿超过2s，不会说就绕过，这也正是Language Lord的方法；
