# Abstract
母语者语言的底层存储并不是细化到具体单词，而是以块、句式来存储。

就跟数据库的底层编码不同一样，以单词作为最终存储结构效率很低，母语者需要构建更多index，这样在调取时候速度就会快很多。

语言在大脑中的基本运作单位，并不等于书面上的最小切分单位，而更常是经过使用频率固化后的语块、构式和情境模板。

这种翻译和理解上的差异，确实会导致母语者和外语学习者在语言底层加工方式上的不同。
对母语者来说，很多表达并不是通过最细粒度成分逐一映射意义来理解的，而是作为整体语义块、构式或场景化模板被直接调用。
以 d'accord 为例，母语者通常不会实时分解为 de + accord 再组合意义，而是把它作为一个整体功能单位来理解和使用。
