# 语体Register
语体（社交语境） 像一个“调节阀”，它不仅决定你选什么词（词汇层），决定你用什么句式（语法层），还决定了你对发音规则的执行严苛程度（语音层）。
### 举例

### 语体分为三个层级
语体 (Registre): 法语分为三个主要层级：

1. Soutenu (雅致): 极其罕见掉音，甚至会增加连诵（如：Il est 必连读）。

2. Standard (标准): 书面和广播用语，/l/ 通常保留。

3. Familier (通俗): 就像你听到的那样，为了效率，大量“弱读音节”被牺牲掉。

逻辑关联 (Logical Linkage): > 这种现象在法语中被称为 "Economie linguistique" (语言经济原则)。当一个词（如 Il）的功能已经由语境清晰表达时，它的发音完整性在“低级别语体”中就会被降低优先级。

TCF考试会全覆盖。

## 语体影响
### 脱落
`通俗语体 (Informal Register): /l/ 在辅音前几乎 100% 脱落（如 Il faut → I'faut）`

`通俗语体 (Informal Register): 当语速极快时，/i/ 作为一个高元音，如果前面有强辅音（如 froid 在某些地区发出的轻微 /d/ 或紧接着前一个词的尾音），第一个 /i/ 可能会发生吞音 (Apheresis)，听起来像 'ci。`
