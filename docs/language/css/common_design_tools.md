# Common Design Tools 
## Design Brief 

```shell
A design brief is a document that outlines the objectives, goals, and requirements of a project.
- The first element is the overview of the project and business. 
- The next key element should be to document the goals and objectives for the project. 
- Another key element would be the target audience. 
- competition and how the project will differentiate itself from the competition.
- Another key element would be the project scope. 
- outlines these constraints is important, One of the challenging aspects about project design is the timescale and budget

```
