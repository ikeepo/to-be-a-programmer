# 输入法

## 如何输入法语
两种方式，一种是长按字母，但是在nvim中无效；另一种是`Option`组合，在`nvim`依然有效。

### Option输入法语

| 目标字符 | 快捷键组合 | 操作说明 |
| :--- | :--- | :--- |
| **é** | `Option` + `e` , 然后按 `e` | 先按住 Option 和 e，松开后再按一次 e |
| **à / è / ù** | `Option` + `` ` `` , 然后按字母 | `` ` `` 键通常在键盘左上角 (Tab 上方) |
| **â / ê / î / ô / û** | `Option` + `i` , 然后按字母 | **记忆法：** `i` 像个小柱子撑起帽子 |
| **ë / ï / ü** | `Option` + `u` , 然后按字母 | **记忆法：** `u` 上方自带两点 |
| **ç** | `Option` + `c` | 直接生成；若需大写 **Ç**，加按 `Shift` |
| **œ** | `Option` + `q` | 常用单词如 *oeuf* 或 *beurre* |
| **« »** | `Option` + `\` / `Shift` + `Opt` + `\` | 法语专用引号 |

> **💡 小贴士：**
> * **输入大写：** 在最后一步按字母键时，配合 `Shift` 键即可。例如输入 **É**：先按 `Option` + `e`，松开后按 `Shift` + `e`。
> * **Neovim 用户：** 如果在终端中无效，请确保终端设置（如 iTerm2）将 Option 键识别为 `Esc+`。
